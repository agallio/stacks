# @agallio/stacks

My personal stack/boilerplate generator.

## List of Stacks

- Next.js
- Next.js (TypeScript)
- Expo 52 + Next.js 14 (Turborepo with yarn)
- Expo 53 + Next.js 15 (Turborepo with yarn)
- Expo 54 + Next.js 15 (Turborepo with pnpm)
- Expo 54 + Next.js 15 App Router (Turborepo with pnpm)

## How-to-Use

```bash
# get it via npx/pnpx
$ npx @agallio/stacks <project-name>
```

Select one of the stack, and it will generate the stack automatically.
